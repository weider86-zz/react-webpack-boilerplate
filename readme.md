# React Boilerplate

React boilerplate - Modern tooling with JavaScript repo repository. Using webpack, babel, jest, testing library eslint, prettier, husky, among others.

## Installation

Use node package manager to install React boilerplate.

```bash
npm install
```

## Run

```bash
npm run dev
```
